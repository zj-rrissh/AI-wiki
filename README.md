# AIWIKI - 个人AI知识库

> 将AI领域原始资料编译为结构化知识的学习库

> ⚠️ **知识库仍在扩展中** - 部分概念尚未完整编译，内容会持续更新

## 目录结构

```
research/
├── raw/           # 原始资料（只读不改）
│   ├── articles/  # 网页剪藏
│   ├── notes/     # 个人笔记
│   └── reports/   # 研报与新闻
└── wiki/          # 编译产物（结构化输出）
    ├── concepts/  # 原子化概念条目
    ├── summaries/ # 逐篇精简摘要
    ├── methods/   # 方法论沉淀
    └── indexes/   # 全局索引
```

## 编译流程

1. **raw/** 中的原始笔记 → 编译摘要到 **wiki/summaries/**
2. 高价值概念 → 沉淀到 **wiki/concepts/**
3. 全局索引 **wiki/indexes/index.md** 自动追踪知识关联

## 技术栈

- **Dataview** - 动态索引查询
- **Tasks** - 行动追踪
- **Git** - 版本控制
